Span Tag :

This is span tag -

  • span tag is aninline container.
  • span tag is used to markup part of text/ content in an element.
  • span tag also helps to apply styles and scripts easily with help of id and classattributes similar to div tag,
  • but div is a block-level element and is an inline element.
  • span can be used inside 'p' tags, to apply different styles to part of the content in the 'p' tag

Is div or span tag supported 'align' attribute?

  • div : Yes, it supported the align attribute.
  • span : No, it did not support the align attribute.
