Div Tag :

This is div tag -1

  • Div tag defines the division/section in the document.
  • It is container of HTML elemennts
  • div is a block-level element, so there will be a line before and after the div tag.
  • div tag cannot be used inside 'p' element.
  • div tag encapsulates HTML elements and divides the HTML document into sections.
  • div tag is used to group the large section of HTML elements together.
