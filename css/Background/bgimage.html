CSS Background Image

The property background-image is used to set the background image of an html element.

By default the image covers the entire element. If the image size is small, by default theimage is repeated to cover the entire element.

The image that has to be set as the background has to be located in the same projectfolder

Syntax: background-image: url(“image path”);
